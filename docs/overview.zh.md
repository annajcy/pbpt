# 概览

PBPT（Physically Based Path Tracer，基于物理的路径追踪器）是一个现代 C++ 渲染器，专注于基于物理的光传输模拟、模块化架构和可复现的离线渲染工作流。

## 特性亮点

- 基于物理的路径追踪管线
- 模块化的场景、材质和光谱子系统
- 基于 CMake + Conan 的构建工作流
- 用于快速实验与对比的示例程序
- Doxygen API 文档生成支持

## 项目结构

- `src/`：核心实现
- `examples/`：可运行的示例程序
- `asset/`：模型、场景和光谱数据
- `test/`：测试套件
- `docs/`：MkDocs 和 API 文档入口

## 典型工作流

1. 使用 Conan 配置依赖和工具链。
2. 使用 CMake 生成构建文件。
3. 构建目标（库、示例、文档）。
4. 运行示例程序以渲染图像。
5. 通过生成的文档查阅和扩展 API。
